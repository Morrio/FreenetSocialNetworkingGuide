Étape 6: (presque) la messagerie instantanée et de chat avec FLIP

LIP est le programme IRC léger Freenet. Il peut être utilisé pour la messagerie instantanée avec des latences de 30 secondes environ..

  1. Le freesite officielcontient un plugin Freenet qui fonctionne sous Windows ou Linux. OS X n'est pas supporté par ce guide pour le moment.

  2. Accédez à la page de configuration du plugin de noeud et entrez la clé pour le plugin FLIP :

    CHK@1G6p~0bUt19pG4abKcTCzmjt7rq941NaCN4Z57vR4lg,teUIaoH8KP~XwJngYROBQ4lT5rwmEaZEjmyvfJb14Y4,AAMC--8/FLIP-plugin-0.1.7.jar

  3. FLIP ne crée pas d'éléments de menu dans l'interface web. Dès le chargement du plugin, il commence à écouter sur l'interface localhost.

  4. Pour utiliser FLIP vous aurez besoin d'un client IRC, idéalement séparé de celui que vous utilisez pour d'autres réseaux IRC. Ce guide va utiliserKVIrc.

    Aller à la page de téléchargement et sélectionnez le package correspondant à votre système.

  5. Après l'installation terminé, exécutez-le.

  6. Choisissez votre langue.

  7. Accepter les termes de la licence.

  8. Choisissez les raccourcis que vous voulez installer.

  9. Maintenant vous êtes prêt à installer réellement KVIrc.

  10. Lorsque l'installation est terminé, cliquez sur «Terminer» pour lancer l'assistant d'installation.

  11. Ce guide suppose que vous n'avez pas de réglage précédent à importer.

  12. Juste au cas où vous avez changé d'avis, acceptez la licence à nouveau.

  13. Les paramètres par défaut devraient suffire ici.

  14. Choisir un pseudo ici. Contrairement aux autres plugins que nous avons utilisées jusqu'ici ce surnom ne sera pas liée à l'identité de votre WoT. Vous pouvez choisir ce que vous voulez ici.

  15. La plupart des gens voudront le thème par défaut, mais si vous préférez couleurs inversées, vous pouvez choisir «minimaliste».

  16. Vous ne voulez probablement pas que KVIrc être le client IRC par défaut parce que ce client peux être utiliser seulement pour Freenet.

  17. La liste des serveurs est peuplé de plusieurs réseaux et serveurs IRC courants. Comme nous n'utiliserons pas l'un d'eux cliquez sur le bouton «New Network».

  18. Remplissez le nom du réseau (FLIP), puis cliquez sur «Nouveau serveur».

  19. Changer le nom du serveur à 'localhost' et cliqué sur le bouton "Avancé".

  20. Sur l'onglet Connexion cochez la case se connecter à ce serveur au démarrage, puis cliquez sur "OK".

  21. Décochez la case «Afficher cette boîte de dialogue au démarrage», car nous avions seulement configuré pour se connecter à KVIrc FLIP au démarrage. Puis cliquez sur "Connect Now" pour commencer.

  22. C'est une autre boîte de dialogue vous n'avez probablement pas envie de voir à chaque fois que vous démarrez le programme. Rejoignez quelques canaux ici et cliquez sur «Fermer». # freenet et # test sont bons à se joindre à un minimum.

    # test contient normalement un bot IRC que vous pouvez utiliser pour tester votre temps de latence.

  23. Si vous voulez voir ce que les autres canaux sont disponibles, cliquez sur «Obtenir la liste des canaux».

  24. Cliquez sur le bouton "Liste des commandes" pour obtenir une liste de tous les canaux FLIP a vu.

  25. Vous ne voulez probablement pas joindre vos chaînes préférées manuellement chaque fois que vous démarrez KVIrc alors allez au menu «Configurer les serveurs de mettre en place le chargement automatique».

  26. Sélectionnez le réseau FLIP et cliquez sur le bouton "Avancé".

  27. Sur l'onglet Canaux Rejoignez ajouter tous les canaux que vous souhaitez vous connecter automatiquement lorsque vous démarrez KVIrc.

Ceci conclut le guide de réseautage social Freenet. A présent, vous avez du courrier sécurisé, micro-blogging, forum et la capacité de la messagerie instantanée.
